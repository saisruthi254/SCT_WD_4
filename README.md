# To-Do Web Application

This project was completed as part of my SkillCraft Technology Internship (Task 04).

It is a simple To-Do Web App that allows users to manage their daily tasks.

## Features

- Add, edit, and delete tasks  
- Mark tasks as completed  
- Add date and time to tasks  
- Data stored using local storage  

## Technologies Used

HTML, CSS, JavaScript
